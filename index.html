You’re using ChromeOS!

ChromeOS is a bit different from Windows, but don’t worry, you can still do almost everything you need.

  • There’s no traditional desktop — your apps run in browser-like windows.
  • Most tasks are done through the Chrome browser or web apps.
  • You can access your files through the Files app, similar to File Explorer on Windows.
  • To Right Click, use two fingers to tap on the trackpad.
